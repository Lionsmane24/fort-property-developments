next-plausible PlausibleProvider usage:
```typescript
import PlausibleProvider from 'next-plausible'

// Wraps body children:
<PlausibleProvider domain="www.FortPropertyDevelopments.com">
  <Nav />
  <main>{children}</main>
  <Footer />
</PlausibleProvider>
```

PlausibleProvider is a server-compatible wrapper. It does NOT require 'use client'.
Domain must match exactly what Dennis registers in Plausible dashboard.

If Plausible account not yet active: PlausibleProvider still renders safely — the script
just won't record hits until the domain is verified in the dashboard. No runtime error.
