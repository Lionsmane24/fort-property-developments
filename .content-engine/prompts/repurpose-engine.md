# Repurposing Engine — 1 Pillar Piece → 8+ Posts

## Master Repurposing Prompt

```
You are the content engine for Fort Property Developments. Take this pillar content and create platform-specific derivatives.

PILLAR PIECE:
"""
[Paste the full pillar content — blog post, video transcript, or long-form write-up]
"""

PILLAR: [Multiplex Development / BC Zoning & Bylaws / Market Intel / Behind the Build]

Create ALL of the following:

---

### 1. LINKEDIN POST
- 200-250 words
- Lead with the strongest insight from the pillar piece
- First person (Dennis)
- End with engagement question
- Include: HOOK, BODY, CTA, COMMENT HASHTAGS

### 2. LINKEDIN CAROUSEL
- 7-8 slides
- Slide 1: hook headline
- Slides 2-6: key points (1 per slide, max 20 words)
- Slide 7: takeaway
- Slide 8: CTA + follow
- Include: slide text + Canva design notes

### 3. TWITTER/X THREAD
- 5-7 tweets, max 280 chars each
- Tweet 1: hook + 🧵
- Each tweet: standalone value
- Last tweet: CTA + follow
- Number format: 1/ 2/ 3/

### 4. TWITTER/X STANDALONE QUOTE
- Single tweet pulling the most quotable line
- Max 280 chars
- Should work completely out of context

### 5. INSTAGRAM CAROUSEL
- 7-8 slides
- Visual-first — describe the design for each slide
- Include full caption (100-150 words + 20 hashtags)

### 6. INSTAGRAM REEL SCRIPT
- 30-60 seconds
- Hook (0-3s) + Body (3-45s) + Close (45-60s)
- Include [TEXT ON SCREEN] and [VISUAL] directions
- Conversational delivery

### 7. TIKTOK SCRIPT
- 15-45 seconds (shorter than Reel)
- Faster pace, more casual
- Trend-aware hook
- Include text overlay directions

### 8. FACEBOOK POST
- 100-200 words
- Community-oriented
- Local angle
- Engagement question at end

---

## FORMATTING RULES:
- Separate each piece with a clear header: "## [PLATFORM] — [FORMAT]"
- Each piece should feel native to its platform — not copy-pasted
- Vary the angle: if LinkedIn leads with data, Instagram should lead with story
- Never repeat the same hook across platforms
- Include Canva design notes for any visual content
```

## Weekly Batch Prompt

```
Generate this week's content for Fort Property Developments.

WEEK OF: [date]

PILLAR PIECES THIS WEEK:
1. [Topic 1 — Pillar: ___]
2. [Topic 2 — Pillar: ___]
3. [Topic 3 — Pillar: ___]

For each pillar piece, generate the full 8-piece derivative set (LinkedIn post, LinkedIn carousel, Twitter thread, Twitter quote, Instagram carousel, Instagram Reel, TikTok, Facebook).

Then add:
- 3 curated news reaction posts (LinkedIn + Twitter + Facebook) based on this week's industry news
- 2 engagement-only tweets (questions, polls, hot takes — no links)
- 2 Instagram Stories concepts (polls, questions, behind-scenes)

Total output: ~30-35 pieces for the week.

Format each with:
- Platform
- Format
- Publish day (Mon-Sun)
- Suggested time
- Full copy
- Visual/design notes if applicable
```
