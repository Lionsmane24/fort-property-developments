# Google Alerts — Setup Guide

## How to Set Up

Go to google.com/alerts → Sign in with your Google account → Create each alert below.

**Settings for all alerts:**
- How often: **As-it-happens** (for breaking news) or **Once a day** (digest)
- Sources: **Automatic** (Google picks best)
- Language: **English**
- Region: **Canada**
- How many: **Only the best results**
- Deliver to: **RSS feed** (then add to Feedly) or **Email**

**Pro tip:** Deliver to RSS feed and add each feed to the matching Feedly board.
