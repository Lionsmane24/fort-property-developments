# Feedly Curation Boards — Setup Guide

## How to Set Up

1. Go to feedly.com → Create account (Pro recommended for AI features)
2. Create the boards below
3. Add the RSS feeds listed under each board
4. Set up Feedly AI to highlight articles matching your keywords

---

## Daily Curation Workflow

1. **Morning scan (5 min):** Check Feedly highlights → star 2-3 articles
2. **Quick reaction:** Pick 1 starred article → use Curated News prompt → create LinkedIn + Twitter post
3. **Weekly deep dive:** Pick the best article of the week → use it as a pillar piece → full repurposing engine
