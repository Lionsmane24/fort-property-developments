# Canva Brand Templates — Setup Guide

## Brand Kit (Set Up First)

In Canva → Brand Kit:

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Fort Charcoal | `#2C2C2C` | Primary backgrounds, text |
| Fort Gold | `#C9A84C` | Accents, highlights, CTAs |
| White | `#FFFFFF` | Text on dark backgrounds |
| Light Gray | `#F5F5F5` | Secondary backgrounds |
| Warm Gray | `#9A9A9A` | Subtle text, dividers |
| Dark Gold | `#A8882E` | Gold variation for hover/press states |

### Fonts
| Usage | Font | Weight |
|-------|------|--------|
| Headlines | Playfair Display | Bold |
| Body / UI | Inter | Regular, Medium, Semibold |
| Accent / Labels | Inter | Semibold, Uppercase, Tracked |

### Logo
- Upload: `fort-logo-reversed.svg` (white on dark)
- Upload: standard logo (dark on light) if available
- Set as default brand logo

## Template 1: LinkedIn Carousel (1080×1350px)

### Create 8-slide master template

**Slide 1 — Cover/Hook**
- Background: Fort Charcoal solid
- Gold accent line (2px) across top
- Headline: Playfair Display Bold, 48px, white, centered
- Subheadline: Inter Regular, 20px, warm gray, centered below
- Fort logo: bottom-right corner, small, white
- Gold dot accent: top-left decorative element

**Slides 2-6 — Content Slides**
- Background: Fort Charcoal
- Slide number: Inter Semibold, 14px, gold, top-left ("02")
- Headline: Playfair Display Bold, 36px, white
- Body text: Inter Regular, 18px, light gray
- Gold accent bar: left side vertical stripe (4px)
- Optional: icon placeholder area (top-right)

**Slide 7 — Takeaway**
- Background: Gold (#C9A84C)
- "KEY TAKEAWAY": Inter Semibold, 14px, charcoal, uppercase
- Takeaway text: Playfair Display Bold, 32px, charcoal
- Feels like a "highlight" slide — stands out from the rest

**Slide 8 — CTA**
- Background: Fort Charcoal
- Fort logo: centered, larger
- "Follow @FortPropertyDev": Inter Semibold, 24px, white
- CTA text: Inter Regular, 16px, gold
- Website: Inter Regular, 14px, warm gray

### Duplicate this template 4 times and pre-label:
1. `Fort Carousel — Zoning Education`
2. `Fort Carousel — Market Data`
3. `Fort Carousel — Development Insights`
4. `Fort Carousel — Behind the Build`

## Batch Creation Workflow

1. Open the relevant template in Canva
2. Duplicate slides as needed
3. Replace placeholder text with content from the prompts
4. Export as PNG (carousel slides) or MP4 (video covers)
5. Upload to Buffer/Later for scheduling

**Time estimate:** 15-20 min per carousel once templates are set up.
**Batch 5 carousels:** ~90 min for a full week of visual content.
